What is HTML?

HTML, or Hypertext Markup Language, is the standard language for creating web pages. It uses tags to structure content and define elements like headings, paragraphs, links, and images. HTML works with CSS and JavaScript to design and add interactivity to web pages.

HTML element

An HTML element is a fundamental component of a web page defined by HTML tags. It comprises the content enclosed within opening and closing tags, specifying the structure and meaning of different parts of the page, such as paragraphs, headings, links, and images.

Tags

The primary building block in html is tags. There is up to 142 HTML tags in total, to name some of the common ones - h1, h2, p, img, header, main, footer. Most of them start with a opening tag and end with a closing tag. The ones without closing tags will be mentioned furtherdown.
