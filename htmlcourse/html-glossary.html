HTML Glossary

A

Used to create hyperlinks, which are clickable links to other pages or locations within the same page. Example:
<a href="myWebsite.com">Visit Site</a>

Defines independent, self-contained content that could be distributed or reused, such as articles, blog posts, or news stories. Example:
<article>...</article>

Used to represent a section of content that is tangentially related to the content around it, such as sidebars or call-out boxes. Example:
<aside>...</aside>

Additional information about HTML elements that provides more details or modifies behavior.

B

The main content of an HTML document, where all visible elements are placed. Example:
<body>...</body>

Used to create clickable buttons within HTML documents. Example:
<button>Click Me</button>

C

An attribute used to assign one or more class names to an element, which can then be targeted by CSS or JavaScript. Example:
<div class="example">...</div>

Used to insert comments in the HTML code that are not displayed in the browser. Example:
<-- This is a comment -->

D

A block-level container used to group elements and apply styles or layout schemes. Example:
<div>...</div>

A declaration at the beginning of an HTML document that specifies the version of HTML used. Example:
<!DOCTYPE html>

E

The fundamental building block of HTML, represented by tags that define the structure and content of a web page. Example:
<p>This is a paragraph</p>

F

Used to create interactive controls for user input, such as text fields, checkboxes, and submit buttons. Example:
<form action="submit">...</form>

H

Represents introductory content or a set of navigational links. Example:
<header>...</header>

Contains meta-information about the HTML document, such as its title, character set, and links to scripts and stylesheets. Example:
<head>...</heder>

Elements used to define headings of different levels, from h1 to h6 Example:
<h1>Biggest heading</h1>

I

Used to embed another HTML page within the current page. Example:
<iframe src="https://www.example.com"></iframe>

Used to embed images in HTML documents. Example:
<img src="image.jpg" alt="Description of image">

Used within forms to create input controls such as text fields, radio buttons, and checkboxes. Example:
<input type="text" name="username">

L

Used to define relationships between the current document and an external resource, such as stylesheets. Example:
<link rel="stylesheet" href="styles.css"

Elements used to create lists. There are two types: Ordered list <ol> and unordered list <ul>, which both then contain list items <li> Example:
<ol><li>...</li></ol>

M

Represents the main content of the body of a document, which is directly related to the central topic of the document. Example:
<main>...</main>

Provides metadata about the HTML document, such as description, keywords, and character set. Example:
<meta charset="UTF-8">

N

Used to define a set of navigational links. Example:
<nav>...</nav>

P

Used to define a block of text as a paragraph. Example:
<p>This is a paragraph.</p>

S

Used to include or reference JavaScript within an HTML document. Example:
<script src="script.js">...</script>

Defines a section in a document, such as chapters, headers, footers, or any other sections of the document. Example:
<section>...</section>

An inline container used to mark up a part of a text or a part of a document. Example:
<span>This is a span.</span>

T

Used to create a table. Example:
<table>...</table>

The markup used to define HTML elements, consisting of an opening tag and a closing tag, with content in between. Example:
<div>...</div>

An element within the head section that defines the title of the document, displayed in the browser's title bar or tab. Example:
<title>My web page!</title>
