Getting started

Webpages aren't just built up from singular tags on their own. Let's examen what it takes to build a web page:

Several things is happening, here's what they do:

  1. <!DOCTYPE html>: this line of code is a old artifact but is still crucial for the HTML document to work. It essentially tell this browser which language this document is.
  2. <html>: The html tag let's the browser know where the html code starts and ends. The tags also has a attribute which tells the browser which language will be used on the website, which in this instance is english.
  3. <head>: The head element contains everything that should be included on the page without showing it in the contant. This could include SEO, style sheets, character declaration and more.
  4. <meta>: The meta tag contains metadata and with the attribute charset set to UTF-8, you'll be able to write in the language and with letters from the majority of human languges.
  5. <title>: The title element does what the name suggests. It sets the title of the page which can be displayed in the browser tabs.
  6. <body>: The body tag contains all content that the page should display including text, images, videoes and much more.

Selecting your editor

To even start writting any code you'll need a text editor. The most common editor is Visual Studio Code or VSCode for short. This will be your main tool for coding and creating websites!

The 5 most common text editors are:

  1. Visual Studio Code
  2. Sublime Text
  3. Atom
  4. Notepadd++
  5. Vim

We highly recommend Visual Studio Code as is is the industry standard, it's free and hos tons of helpfull plugins for whatever you might need!
