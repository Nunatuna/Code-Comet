Syntax

Like every other programming language, there's a set of rules, and it's not exception for Javascript.

It's a long list for Javascript but we will start off by defining the two types of values Javascripts syntaxes hold.

Literals

Also know as fixed values - So a value that won't change and is written into the code.
Here's a list of the 3 most common literals in Javascript:

  1. Numeric Literals - These represent numbers and can be written with or without decimals.
  2. String Literals - These represent text and must therefore be wrapped in double- (" ") or single (' ') quotations.
  3. Boolean Literals - These represent what's called "Logical Values" which can be "true" or "false".

Variables

Variables in Javascript are used to store data that can be referenced or manipulated.

Javascript uses the keywords - var, let and const to declare a variable.
After the keywords a = will be used to assign value to variables.
