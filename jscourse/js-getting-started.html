Like CSS, there's 3 ways to impliment Javascript into your HTML.

  1. Inline
  2. Internal script
  3. External script

Much like CSS, it's concidered best practice to use external scripting. This is how it's done:

Like CSS you link to the external Javascript document but instead of being in the link element, it's got it's own script tag which will link to the document. The script tag can be placed in the head tag but can also be placed in the body tag.
