Colors

Colors can be specified using different methods using, predifines names, RGB, HEX, HSL, RGBA or HSLA.

Predifined names

In CSS there's predefined color names which makes it quick and easy to select a color if you don't have a specific color in mind.

There's a total of 140 predifined color names.
These colors can simply be chosen by writting a color name as the value on a color property. Here's a few examples:

Yellow
Orange
Red
Green
Blue
Purple

RGB

An RGB color consists of 3 specific color values of Red, Green and Blue.

The value of each color goes from 0 - 255 and then mixes the colors to creat a final color. The example for purple would be:

background-color: rgb(128, 0, 128);

Hex

Another common value to define a color is hexadecimal. And looks like this:

#rrggbb

It works like this. Much like the RGB the Hexadecimal is divided into 3 color values, Red (rr), Green (gg) and Blue (bb). It goes from "00" to "ff".

Here's how it works in practice:

#ffff00
#ffa500
#ff0000
#00ff00
#0000ff
#800080

HSL

HSL stands for Hue, Saturation and Lightness.

Using HSL, developers have more precise control over colors, making it easier to create and manipulate color schemes in web design.
