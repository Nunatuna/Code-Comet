Box Model

In HTML, everything has a box around it.
With CSS you can control these boxes and change how they look and act.

The CSS box model consists of 4 things:

  1. Content - The content you add to the box, such as images and text
  2. Padding - The padding creates space around the content inside the box
  3. Border - The border makes a line around the content
  4. Margin - The margin creates space around the box
