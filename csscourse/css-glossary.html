CSS Glossary

A

A method to add motion and changes over time in CSS properties. Example:
@keyframes example { from {background-color: red;} to {background-color: yellow;} }

B

Example:
background: #ffffff url('image.jpg') no-repeat right top;

Used to define the borders around an element, including style, width, and color. Example:
border: 1px solid black;

Refers to the box properties of an element, including margin, border, padding, and the content itself. Example:
margin: 10px; border: 1px solid black; padding: 10px;

Example:
box-shadow: 10px 10px 5px #888888;

C

An attribute used to assign style rules to one or more HTML elements. Example:
<div class="example">...</div>

Used to define the color of text in an element. Example:
color: blue;

D

Defines how an element is displayed on a webpage (block, inline, flex, grid, etc.). Example:
display: block;

F

A layout method used to arrange elements in a flexible container. Example:
display: flex;

Used to place an element to the left or right, allowing text and inline elements to wrap around it. Example:
float: left;

Properties related to fonts, such as type, size, weight, and style. Example:
font-family: Arial, sans-serif; font-size: 16px;

G

A layout method used to create complex and responsive grid-based layouts. Example:
display: grid;

H

Sets the height of an element. Example:
height: 100px;

I

A unique identifier used to assign style rules to a specific HTML element. Example:
<div id="example">...</div>

M

Sets the outer space around an element. Example:
margin: 20px;

P

Sets the inner space within an element between the content and its border. Example:
padding: 10px;

Used to specify how an element is positioned in the document (static, relative, absolute, fixed, sticky). Example:
position: absolute;

A keyword added to a selector that specifies a special state of the selected element. Example:
a:hover { color: red; }

Used to style specific parts of an element, such as the first letter or first line. Example:
p::first-line { font-weight: bold; }

R

An approach to web design that makes webpages adapt and function well on all devices and screen sizes. Example:
@media (max-width: 600px) { .example { font-size: 12px; } }

S

Used to select the HTML element(s) to be styled. Example:
p { color: red; }

T

Used to add shadow effects to text. Example:
text-shadow: 2px 2px 5px gray;

Used to add a smooth transition between two different states of an element. Example:
transition: background-color 0.5s ease;

W

Sets the width of an element to a specific size. Example:
width: 20px;
