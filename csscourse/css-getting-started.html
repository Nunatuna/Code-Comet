Getting started

CSS documents have a very simple structure and consists of just 3 parts: Selector, Property and Value.
Let's get into them one by one.

CSS Selector

CSS selectors will point to an element inside the HTML document it's linked to to change it's style and apprearance.

Not all selectors are created equally and there is therefore a hierarchy, here's how they line up starting from priority one and downwards:

  1. ID selectors: ID selectors is definied by you and has the highest priority when styling, meaning it will overwrite anyother selectors that may be styling an element. CSS Property

Properties are the actual things that we can change on elements and there's a whole bunch of them, but we'll only be covering some of the most common ones you need to begin with:

  1. Color : Changes the color of an element.
  2. Background-color : Sets the background-color to whatever color you wish.
  3. Font-size : Changes the size of the font.
  4. Font-family : Changes the font-family.
  5. Font-weight : Changes the boldness of a font.
  6. Margin : Sets the margin around an element to create space around.
  7. Padding : Sets the padding inside an element to create space inside.
  8. Border : Will draw a border around the element in any width or color.
  9. Width : Sets the width to a specific width.
  10. Height : Sets the height to a specific height.

CSS Value

CSS values are the settings used to style HTML elements. They determine how properties like color, size, position, and more are applied. These values can be specific (e.g., 10px for a size) or predefined (e.g., red for a color). Understanding and choosing appropriate CSS values is essential for designing visually appealing and functional web pages.
