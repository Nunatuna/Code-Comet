Fonts

Time to give your website personality!
CSS fonts shape the appearance of text on the web, adding style and personality. They range from elegant serifs to modern sans-serifs, making your website unique and engaging. The right font choice enhances both visual appeal and readability.

To begin we need to chose a font family, which can be done multiple ways.
Some fonts are built into CSS and therefor always a safe choice - They're called "web safe fonts" since the browser will always be able to display them.

If the built in ones won't do, then there's a selection of website hosting font families, here's a few places to find them:

  • Google Fonts
  • FontSpace
  • Font Squirrel
  • DaFont
  • Adobe Fonts

We recommend Google Fonts since all the fonts are open source and freely available and is also an industry standard.
